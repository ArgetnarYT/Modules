# Modules for Lua scripts in Roblox
##Using
To use this module library, you need to add this line at the beginning of your code:
```lua
local ArgetnarLib = loadstring(game:HttpGet("https://argetnarhub.tk/raw/Modules.lua"))()
```
## Basic
- Teleport `tp(x,y,z)`
- Fly on `Fly(true)`
- Fly off `Fly(false)`
- Speed `Speed(v)`
- Gravity `Gravity(v)`
- Kick `Kick(reason)`
- BTools `BTools()`
- Noclip `noclip()`
- Clip `clip()`
- AntiAfk `AntiAfk()`
- ESP on `Esp(true)`
- ESP off `Esp(false)`
